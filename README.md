# ArchAI
ArchAI is an AI orchestration platform for automated architectural design. It uses a "Council of Engineers" (8 specialized agents) to debate system stacks, security, and costs in real-time via SSE. Moving from conceptual input to full-stack prototypes, it optimizes performance using multi-model routing (Claude 3.5/4.5/4.6/gemini/gpt-5.4).
